transform插件

发表于 2017-08-07

andyTransform.js

该插件是一个用于通过调用transform的matrix方法来实现物体动画效果的插件。比如，我们之前会使用css的transform的translate，rotate，scale等方法来实现过渡动画，但是每次都是写在css中，或者通过给style对象添加transform属性来实现。该插件就是通过js来控制动画效果，比如，我们想让一张图片水平移动100px，我们只需要这样：

而且效果是可以叠加的，比如，我们想让图片水平方向移动100px，垂直方向移动100px，并且转动40deg。我们就可以这样：

是不是很简单呢？同时我们还可以配合自己所需的动画效果来进行拓展，比如：下面的运动是匀速的，每个16.7毫秒执行一次。当然你可以根据自己的逻辑实现更加丰富的动画。

为什么会写这个插件？

写这个插件的目的源于最近公司的一个需求，移动版上面的滑动效果，当然网上有很多不错的插件可以实现该效果，但是对于我自己来说，更喜欢的就是自己写。

写这个插件遇到的问题？

其实主要的问题就是矩阵的应用，说实话，我是学文科的，转行当程序员，看到代数相关确实不懂，所以网上找资料稍微学习了下。
